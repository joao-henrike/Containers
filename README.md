## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Open%20Book.png" width="35" /> Como usar
```bash
# Clone o repositório
git clone https://github.com/joao-henrike/Containers.git

# Acesse o container desejado
cd Containers/<nome-do-container>

# Siga o README.md interno do container
cat README.md
```
